Workflow Studio brings a visual, node-based automation builder into the
Odoo backend.

Design workflows on a drag-and-drop canvas, configure triggers, and
execute them with the built-in workflow engine without leaving Odoo.

Main capabilities include:

- visual canvas editor for workflow modeling
- execution history and run inspection
- manual, schedule, webhook, and record-event triggers
- node-based logic such as HTTP requests, branching, loops, and variable
  handling
- extensible execution hooks for advanced automation customization like
  ``queue_job`` integration

Installation
============

Install the ``workflow_studio`` addon like any standard Odoo addon:

1. Add this repository to the ``addons_path`` of your Odoo 18 instance.
2. Update the Apps list.
3. Install **Workflow Studio** from the Apps menu.

The module depends on core Odoo modules ``base``, ``web``, ``mail``, and
``bus``.

For local development, upgrade the addon after pulling changes so
regenerated assets and metadata stay in sync with the codebase.

Usage
=====

After installation:

1. Open **Workflow Studio** from the backend menu.
2. Create a workflow and add trigger and action nodes on the canvas.
3. Configure node data from the side panel.
4. Save the workflow and run it manually, or activate an automated
   trigger.

Expressions are evaluated only inside ``{{ ... }}`` blocks.

For a quick smoke test, start with a manual trigger workflow and add a
simple action node before moving on to scheduled or event-driven
automation.
