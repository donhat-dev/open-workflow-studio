Workflow Studio brings a visual, node-based automation builder into the Odoo backend.

Design workflows on a drag-and-drop canvas, configure triggers, and execute them with the built-in workflow engine without leaving Odoo.

Main capabilities include:

- visual canvas editor for workflow modeling
- execution history and run inspection
- manual, schedule, webhook, and record-event triggers
- node-based logic such as HTTP requests, branching, loops, and variable handling
