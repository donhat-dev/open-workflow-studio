Design Tokens

Two parallel token sets. w-* (Current) uses Inter + Bootstrap-derived values. wf-* (Future) now follows extracted NAAI cues: Space Grotesk across UI, cobalt-blue primary, sharper radii, border-led surfaces, mono metadata, and the editorial framed/grid grammar pulled from the landing page.
