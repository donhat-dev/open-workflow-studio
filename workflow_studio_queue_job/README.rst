This addon extends Workflow Studio with optional ``queue_job``
integration.

When enabled on a workflow, automated trigger executions are intercepted
and dispatched asynchronously through ``queue_job``, while manual
launches and test-mode executions remain synchronous.

Installation
============

To use this addon:

1. Install ``workflow_studio``.
2. Install the OCA ``queue_job`` addon in the same Odoo instance.
3. Install **Workflow Studio Queue Job** from the Apps menu.

Usage
=====

Open a workflow and enable the **Run in Queue** option when you want
automated trigger launches to be processed asynchronously.

Manual executions from the UI and trigger test calls remain synchronous
so operators can get immediate feedback.
